# COOC Website

COOC 의 공식 웹사이트 프로젝트입니다.
이 프로젝트는 Next.js 기반으로 제작된 **에이전시 랜딩 웹사이트**입니다.

🔗 Live Site
https://cooc.agency/


# Tech Stack
### Frontend
* Next.js
* React
* TypeScript
* TailwindCSS

### Icons
* lucide-react

### Deployment
* Vercel


# Features
* 반응형 웹 디자인
* 서비스 소개 페이지
* 이벤트 / 프로세스 소개
* 문의(Contact)
* SEO 기본 설정
